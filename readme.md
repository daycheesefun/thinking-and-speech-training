# 思维与发言教练 · 热点话题训练助手

>这是一个记录自己ai学习成果的demo， 一个用来帮助训练“热点话题概括总结与发言能力”的 AI 教练，也是我第一次零代码体验 Coze 的作品。

## 🎯 两个目的

1. **训练自己的发言能力**  
   针对每天的热点话题，用结构化的思维模型（抽象阶梯法、系统归因法、表达升级、金句收尾）进行概括、总结和观点提炼，逐步提升发言层次。

2. **试用 Coze 平台**  
   作为一名文科生、零编程基础，想试试能不能用纯自然语言方式，搭建一个真正有用的 AI Agent。

## 🧩 制作路径（零代码）

> 以下是我在 Coze 上的完整操作步骤，每一步都不需要写代码。

| 步骤 | 操作 |
| :--- | :--- |
| **1** | 进入 **扣子编程** → 选择 **智能体开发** |
| **2** | 点击 **创建智能体**，输入名称（如“思维与发言教练”）和功能介绍 |
| **3** | 在打开的界面**左侧**，找到 **“人设与回复逻辑”**，将写好的 Prompt（Markdown 格式）粘贴进去 |
| **4** | 在界面**中间**，选择搜索引擎（如“必应搜索”），让教练可以联网获取热点话题 |
| **5** | 在界面**右侧**的 **“预览与调试”** 区域，直接与新建的 Bot 对话，开始训练 |

## 🧠 训练方式（交互式分步）

教练不会一次性输出所有结果，而是**一步一步引导你**：

1. 你发送一个热点话题（或让教练先帮你找热点）
2. 教练用“抽象阶梯法”连续追问，帮你提炼本质
3. 教练用“系统归因法”帮你找关键杠杆点
4. 教练要求你用“不是…而是…”“三词口诀”等技巧升级表达
5. 教练和你一起总结一句金句

> 全程像真教练一样：提问 → 等你回答 → 评价反馈 → 下一步。
